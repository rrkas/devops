# Docker compose commands

## **docker-compose up**
builds and runs the multi container application
> `docker-compose up`

**with build**
> `docker-compose up --build
**detached mode**
> `docker-compose up -d`

## **docker-compose down**
stops and removes containers
> `docker-compose down`

## **docker-compose ps**
shows status and details of containers of a docker-compose.yaml
> `docker-compose ps`
